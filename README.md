# FileSharer
FileSharer is a application (API + website) to store and load files using unique tokens.  
Swagger UI: https://fs.konkov.xyz

## Requirements
1. Docker
2. Docker-compose v1.29.1

## Build
It builds docker image with application.
```bash
make build
```

## Run
It runs services.
```bash
make run
```

## All
Clean, build, run.
```bash
make all
```
